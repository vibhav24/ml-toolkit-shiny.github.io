INSTRUCTIONS

Here are some instructions for the users to use our app.

Step:- 1

This is the "Data" tab in which you have to upload your dataset.


Step:- 2

After browsing and uploading your dataset, the data tab appears like this.


Step:- 3

The uploaded dataset can be explored using the explore tab and the options available in it. There are various actions that can be performed on the dataset. Here, summary is selected initially, so you can view the brief summary of your dataset.


Step:- 4

Data visualization is the graphical representation of information and data. By using visual elements like Histogram, Boxplot, etc. Data visualization tools provide an accessible way to see and understand trends, outliers, and patterns in data.
Below are the exaples for histogram plot and scatter plot.
(a) Histogram plot

(b) Scatter plot


Step:- 5

Next is "Rund and Tune Model".It consists of various algorithms like linear Regression, Regularized Regression, Decision tree, Random Forest, Gradient Boosting, Extreme Gradient Boosting, Support vector machine. It also has various parameters like Choosing the target variable, selecting the metric, tuning parameters.


After uploading the training data, user needs to select the appropriate target variable and the modelling technique that is suitable for the dataset. It will generate the summary of the model which is used for analysis of the model.The model that is built using the training data is used for test data and get the accuracy of the algorithm.



Step:- 6

Now in "prediction " tab, we are going to upload the testing data.We obtain the predictions for the test data with respect to the actual values for the respective column by using the model that we have built.
